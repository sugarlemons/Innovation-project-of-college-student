## 流媒体服务器自适应传输优化
AI视频软件是一款基于区块链思想，通过数据挖掘分析提供服务的智能化视频播放系统。本项目下一步计划围绕该平台，实现平台内视频利用语音识别自动生成字幕功能。自动生成字幕包含两种，一种是实时生成字幕，一种是在用户观看视频前预生成字幕。用户可以根据个人喜好选择字幕的生成方式。
